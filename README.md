# Applied Cloud Programming - Coursework Repository

This repository contains all practical coursework and tutorial exercises for the Applied Cloud Programming (ACP) module at the University of Edinburgh.

## Structure

* **Coursework_0:** Introductory tutorials, Docker basics, and Spring Boot demos (Tutorial 1).
* **Coursework_1:** "The Universal Connector" - A microservice integrating PostgreSQL, AWS S3, and DynamoDB (Current Project).
* **Coursework_2:** *Coming Soon...*

## Technology Stack
* Java 17+ / Spring Boot
* Docker & Docker Compose
* PostgreSQL
* Localstack (AWS Simulation)
